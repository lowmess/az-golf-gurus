# azgolfgurus.com

## Installing

```bash
git clone https://github.com/lowmess/az-golf-gurus.git
cd az-golf-gurus
npm i
```

## Credentials

The following keys should be present in your environment to build/develop the site. In non-production environments `dotenv` will be ran.

- `CONTENTFUL_SPACE_ID`
- `CONTENTFUL_ACCESS_TOKEN`
- `YOUTUBE_CHANNEL_ID`
- `YOUTUBE_ACCESS_TOKEN`

## Building

```bash
npm run build
```

## Developing

```bash
npm run start
```

## Colophon

- **Generator:** [Gatsby](https://www.gatsbyjs.org/)
- **CMS:** [Contentful](https://www.contentful.com/)
